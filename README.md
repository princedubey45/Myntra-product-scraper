# Myntra-product-scraper
A Myntra product scraper is a web scraping tool that automatically extracts structured product data from the Myntra e-commerce site, such as item titles, prices, brands, descriptions, images, ratings, and availability. It’s used for market research, price tracking, trend analysis, and competitive insights.
Myntra Product Scraper

Welcome to the Myntra Product Information Scraper! This Jupyter Notebook project, developed by Prithviraj, features a robust Python script that extracts detailed product information from Myntra’s website.

How It Works:

Provide Product Details: Enter the product name and the number of pages you want to scrape.

Start Scraping: Click the "Scrape Data" button to begin the process. The script automatically navigates through Myntra’s listings and collects key details such as brand name, product title, price, and ratings.

Data Processing: The raw data is cleaned and enhanced by extracting additional fields like original price, discounted price, discount percentage, and total number of reviews.

Export Data: The processed dataset is saved as an Excel file named myntra_cleaned_data.xlsx for easy access and further analysis.

Analyze Insights: Use the generated dataset to explore pricing trends, compare products, and gain valuable market insights.

You can customize the script to scrape different product categories or extend it with additional features. Happy Coding and Scraping! 🚀
